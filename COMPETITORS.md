# Competitor Analysis

## Direct Competitors

### 1. VisaHQ (visahq.com)
**Model:** Full-service visa processing agency

| Aspect | Details |
|--------|---------|
| **What they do** | End-to-end visa processing â€” you mail them documents, they handle embassy |
| **Pricing** | Service fee + government fee ($100-300+ total) |
| **Strengths** | Physical presence (offices in LA, NY, etc.), established since 2003 |
| **Weaknesses** | Expensive, overkill for DIY applicants |
| **Differentiation** | We're info/validation only, not processing |

### 2. iVisa (ivisa.com)
**Model:** Online visa processing + e-visas

| Aspect | Details |
|--------|---------|
| **What they do** | Process visa applications, specializes in e-visas |
| **Pricing** | Processing fee on top of govt fee |
| **Strengths** | Fast (claims 24h for some), AI-powered checks, 24/7 support |
| **Weaknesses** | Only handles destinations with e-visas well |
| **Differentiation** | We're free info/validation, not paid processing |

### 3. Atlys (atlys.com)
**Model:** Modern visa processing platform

| Aspect | Details |
|--------|---------|
| **What they do** | Visa processing with mobile-first UX, 120+ destinations |
| **Pricing** | "Guaranteed on time" with refund policy |
| **Strengths** | Slick UX, 5M+ visas processed, VC-funded ($4.25M) |
| **Weaknesses** | Processing-focused, not validation |
| **Tools they offer** | Mock interview tool, photo creator, appointment checker |
| **Differentiation** | They process; we validate before you apply |

### 4. TravelDoc (traveldoc.aero)
**Model:** Requirements checker (B2B + B2C)

| Aspect | Details |
|--------|---------|
| **What they do** | Real-time entry requirements check |
| **Scale** | 1 billion checks/year, 40+ airline customers |
| **Strengths** | Industry standard (used by airlines), covers transit |
| **Weaknesses** | Basic info, doesn't show detailed document checklist |
| **Partnership** | Sends users to iVisa for applications |
| **Differentiation** | They show IF you need visa; we show WHAT documents you need |

## Free Information Resources

### 5. Travel.State.Gov (Official)
- **What:** Official US government source
- **Pros:** Authoritative, always current
- **Cons:** Scattered across 200+ pages, hard to navigate, no personalization

### 6. VisaGuide.World
- **What:** Free visa requirements database
- **Pros:** Clean UI, covers many countries
- **Cons:** Generic info, not document-level detail

### 7. Sherpa (joinsherpa.com)
- **What:** Travel restrictions/requirements API + widget
- **Model:** B2B (sells to airlines/OTAs)
- **Relevance:** Competitor for the data layer, not consumer product
